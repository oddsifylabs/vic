// Setup

Running in 60 seconds.

No cloud account. No installer wizard. No subscription portal. Node.js and a folder.

01
Download & extract

Download the ZIP. Extract to a USB drive or any folder on Windows or Linux. Nothing to install globally — all dependencies stay inside the folder.

02
Start the server

Double-click start.bat on Windows or run bash start.sh on Linux. First run installs Node dependencies automatically. VIC opens in your browser.

03
Add API keys

Go to Settings. Add your Odds API key (500 free requests/month to start) and Claude API key (pay-as-you-go, ~$2–5/month typical). ESPN, weather, and SAO scraper are completely free — no key needed.

04
Hit ⚡ Run Legs

VIC fetches live odds, loads injuries, checks public %, runs the AI model, and returns ranked edges with full thesis and risk notes — in under 30 seconds.

// Privacy

Your data. Your machine.

No cloud sync. No analytics. No account. VIC stores everything in a local folder — and nothing else ever knows your bets exist.

🔒 100% Local Storage

All bets, keys, and logs live in a data/ folder on your drive. Nothing is transmitted to any server. Works offline for everything except live odds.

🖥 Runs On Your Machine

VIC is a local Node.js app — not a SaaS product. No login, no account, no dashboard you don't control. Shut it off and nothing persists anywhere else.

📂 USB Portable

Extract to a USB drive and VIC runs on any Windows or Linux machine. Your bets, keys, and history travel with you. No re-setup, no re-login.
