// WHAT IS CLV?
DEFClosing Line Value = how your odds compare to the line at game time
POSITIVEYou got +105, line closed -110 → you beat the close → sharp
NEGATIVEYou got -110, line closed +105 → you got worse → square
// TARGETS
SHARP+1.5 to +3 CLV average = consistent edge
NEUTRAL0 to +1.5 CLV = breakeven skill
SQUARENegative CLV average = losing long-term regardless of results
// HOW TO CALCULATE
FORMULACLV = (Your Implied %) − (Closing Implied %)
EXAMPLEYou bet -115 (46.5% impl), closed -125 (55.6% impl) → CLV = −9.1
AUTOAUTO-CAPTURE fetches the closing line from Odds API at game time
SNAPBest — odds auto-snapped when bet was logged or via 📸 SNAPSHOT
FINALGood — fetched from scores API after game ended
PREOK — captured before tip-off via AUTO-CAPTURE
MANUALEnter closing odds yourself from your sportsbook
